üîç Glossary

constructor (__init__): Special method that runs automatically when a new object is created.

super(): A built-in function used to call methods of the parent class.

inheritance: A way to pass methods and properties from one class (parent) to another (child).

polymorphism: When a method behaves differently based on the object calling it.

method: A function defined inside a class that acts on its data.

encapsulation: Restricting access to certain internal parts of an object to protect data.

abstraction: Hiding complex implementation details and showing only the essentials.

@abstractmethod: A decorator that defines a method as abstract (must be implemented in child classes).

ABC (Abstract Base Class): A base class from Python‚Äôs abc module used to define abstract classes.

private variable (__name): Variable hidden from outside access, using double underscore naming.

getter/setter: Methods used to read (get) or update (set) the value of private attributes.
