Your First Tkinter Application

Let's start with the simplest possible Tkinter program that creates an empty window:

basic_window.py 
import tkinter as tk  # Import the Tkinter module

# Create the main application window
window = tk.Tk()
window.title("My First GUI")

# Start the application
window.mainloop()

Key Concepts

Tk() - Creates the main window of your application. This is the foundation of all Tkinter programs.

mainloop() - Starts the event loop that keeps your application running and responsive to user interactions.

Building a Complete Form Application

Let's create a practical user registration form with validation:

registration_form.py 
import tkinter as tk
from tkinter import messagebox

class RegistrationForm:
    def __init__(self, master):
        self.master = master
        master.title("User Registration")
        master.geometry("400x300")
        
        # Create form frame
        form_frame = tk.Frame(master, padx=20, pady=20)
        form_frame.pack(fill=tk.BOTH, expand=True)
        
        # Form fields
        tk.Label(form_frame, text="Full Name:").grid(row=0, column=0, sticky=tk.W, pady=5)
        self.name_entry = tk.Entry(form_frame, width=30)
        self.name_entry.grid(row=0, column=1, pady=5)
        
        tk.Label(form_frame, text="Email:").grid(row=1, column=0, sticky=tk.W, pady=5)
        self.email_entry = tk.Entry(form_frame, width=30)
        self.email_entry.grid(row=1, column=1, pady=5)
        
        tk.Label(form_frame, text="Password:").grid(row=2, column=0, sticky=tk.W, pady=5)
        self.password_entry = tk.Entry(form_frame, width=30, show="*")
        self.password_entry.grid(row=2, column=1, pady=5)
        
        # Submit button
        submit_btn = tk.Button(
            form_frame, 
            text="Register", 
            command=self.submit_form,
            bg="#4CAF50",
            fg="white",
            padx=10,
            pady=5
        )
        submit_btn.grid(row=3, column=1, pady=10, sticky=tk.E)
    
    def submit_form(self):
        # Get form data
        name = self.name_entry.get()
        email = self.email_entry.get()
        password = self.password_entry.get()
        
        # Simple validation
        if not name or not email or not password:
            messagebox.showerror("Error", "All fields are required!")
            return
        
        # Show success message
        messagebox.showinfo(
            "Success", 
            f"Registration successful!\nName: {name}\nEmail: {email}"
        )

# Create and run the application
if __name__ == "__main__":
    root = tk.Tk()
    app = RegistrationForm(root)
    root.mainloop()

Advanced Concepts in This Example

  • Object-Oriented Approach: Using classes to organize your GUI code
  • Grid Layout: Precise widget positioning with rows and columns
  • Validation: Checking user input before processing
  • Dialogs: Using message boxes for user feedback
  • Password Fields: Secure input with character masking
