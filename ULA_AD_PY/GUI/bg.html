1. Understanding Windows and Widgets

What is a Window?

A window is the main container where all GUI elements (widgets) are placed.

What are Widgets?

Widgets are interactive elements like buttons, labels, and text inputs that make up your interface.

basic_window.py 
import tkinter as tk  # Import the Tkinter module
window = tk.Tk()      # Create the main window
window.mainloop()     # Start the application
Line Explanation
import tkinter as tk Imports the Tkinter library and gives it an alias (tk) for easier use.
window = tk.Tk() Creates the main application window.
window.mainloop() Starts the GUI event loop, keeping the window open.

2. Basic Layout Management

Three Ways to Position Widgets

.pack()

Automatically places widgets in available space

.place()

Positions widgets at exact (x,y) coordinates

.grid()

Arranges widgets in rows and columns (like a table)

Example: Using .pack()

pack_example.py 
import tkinter as tk

window = tk.Tk()
window.title("Layout Example")

# Using pack()
label1 = tk.Label(window, text="First Label").pack()  # Top
label2 = tk.Label(window, text="Second Label").pack() # Below first label

window.mainloop()
Line Explanation
window.title("Layout Example") Sets the window title.
label1 = tk.Label(...).pack() Creates a label and packs it at the top.
label2 = tk.Label(...).pack() The second label appears below the first.

üí° Pro Tip

.pack() is great for simple layouts, but for more control, learn .grid() which lets you position widgets in rows and columns.

Example: Creating a Simple Form

simple_form.py 
import tkinter as tk

window = tk.Tk()
window.title("Simple Form")

# Label
name_label = tk.Label(window, text="Enter Name:").pack()

# Entry (Text Input)
name_entry = tk.Entry(window).pack()

# Button

window.mainloop()

Code Explanation

Line Explanation
name_label = tk.Label(...).pack() Creates a label for user input.
name_entry = tk.Entry(...).pack() Adds a text input field.
submit_button = tk.Button(...).pack() Adds a clickable button.

4. Simple Event Handling

Making Buttons Do Something

Use command= to define what happens when a button is clicked.

Example: Button with Action

button_event.py 
import tkinter as tk

def button_click():
    print("Button was clicked!")

window = tk.Tk()
window.title("Event Handling Example")

button = tk.Button(
    window,
    text="Click Me",
    command=button_click  # Calls button_click() when clicked
).pack()

window.mainloop()
Line Explanation
def button_click(): Defines a function to run when the button is clicked.
command=button_click Connects the button to the function.

‚ö†Ô∏è Important Note

Don't include parentheses when passing a function to command=. Write command=button_click not command=button_click().

5. Complete Beginner Example

A Simple GUI with All Basics

greeting_app.py 
import tkinter as tk

def greet():
    name = name_entry.get()
    greeting_label.config(text=f"Hello, {name}!")

window = tk.Tk()
window.title("My First GUI")
window.geometry("300x200")

# Label
tk.Label(window, text="Enter your name:").pack()

# Entry (Text Input)
name_entry = tk.Entry(window)
name_entry.pack()

# Button
tk.Button(window, text="Greet", command=greet).pack()

# Output Label (initially empty)
greeting_label = tk.Label(window, text="")
greeting_label.pack()

window.mainloop()

Key Features

  • Text input field to get user's name
  • Button that triggers a greeting
  • Dynamic label that updates with the greeting
  • Window with fixed size (300x200 pixels)

How It Works

  1. User enters name in the text field
  2. Clicks the "Greet" button
  3. The greet() function runs
  4. Function gets the name and updates the label
