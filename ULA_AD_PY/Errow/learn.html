ZeroDivisionError

This error occurs when a number is divided by zero. In Python, this is not allowed and will result in a crash unless handled properly.

try:
  result = 10 / 0
except ZeroDivisionError:
  print("You can't divide by zero!")

IndexError

This error occurs when you try to access an index that doesn't exist in a list or tuple.

try:
  my_list = [1, 2, 3]
  print(my_list[5])
except IndexError:
  print("Index out of range!")

TypeError

This error occurs when you perform an unsupported operation between different data types.

try:
  result = '5' + 3
except TypeError:
  print("Cannot add string and integer")

ValueError

This error occurs when a function receives a value of the right type but inappropriate value.

try:
  num = int("abc")
except ValueError:
  print("Invalid literal for int()")

SyntaxError

This error occurs when the code written is not syntactically correct.

# Invalid Syntax
# print("Hello"

# Corrected
print("Hello")

NameError

This error occurs when you use a variable that has not been defined.

try:
  print(undeclared_variable)
except NameError:
  print("Variable is not defined")
