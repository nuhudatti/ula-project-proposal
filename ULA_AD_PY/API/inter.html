API Authentication

Most production APIs require authentication to protect sensitive data. The NITDA ULA API uses JWT (JSON Web Tokens) with military-grade encryption (AES-256) for secure authentication.

NITDA Security Standard:

All API communications are encrypted end-to-end to protect Nigeria's digital education infrastructure.

Authentication Flow

  1. 1 Get credentials from NITDA Developer Portal
  2. 2 Request access token from auth endpoint
  3. 3 Include token in Authorization header
  4. 4 Refresh token when expired
Python Example 
import requests

# Step 1: Get access token
auth_url = "https://api.ula.nitda.gov.ng/v1/auth/token"
auth_data = {
    "client_id": "your_client_id",
    "client_secret": "your_client_secret",
    "grant_type": "client_credentials"
}

response = requests.post(auth_url, json=auth_data)
access_token = response.json()["access_token"]

# Step 2: Use token in requests
api_url = "https://api.ula.nitda.gov.ng/v1/protected-data"
headers = {
    "Authorization": f"Bearer {access_token}"
}

response = requests.get(api_url, headers=headers)
print(response.json())

Error Handling

Proper error handling makes your API integrations more robust and user-friendly. Always anticipate and gracefully handle potential errors.

Common API Errors

4xx Client Errors

Your application sent invalid data or requests

5xx Server Errors

The API server encountered a problem

Network Issues

Connectivity problems or timeouts

Python Error Handling 
import requests
from requests.exceptions import RequestException

try:
    response = requests.get(
        "https://api.ula.nitda.gov.ng/v1/data",
        headers={"Authorization": "Bearer your_token"},
        timeout=10  # 10 second timeout
    )
    
    # Check for HTTP errors
    response.raise_for_status()
    
    # Process successful response
    data = response.json()
    
except requests.HTTPError as http_err:
    print(f"HTTP error occurred: {http_err}")
    if response.status_code == 401:
        print("Please authenticate first")
    elif response.status_code == 404:
        print("Resource not found")
    else:
        print(f"Status code: {response.status_code}")
        
except RequestException as req_err:
    print(f"Request failed: {req_err}")
    
except ValueError as json_err:
    print(f"Failed to parse JSON: {json_err}")
    
except Exception as err:
    print(f"Unexpected error: {err}")

Working with JSON

JSON (JavaScript Object Notation) is the standard format for API requests and responses. It's lightweight, human-readable, and easy to parse.

JSON Basics

  • 1 Key-Value Pairs: {"name": "value"}
  • 2 Arrays: ["item1", "item2"]
  • 3 Nested Objects: {"user": {"name": "Aliyu"}}

Python JSON Handling

  • 1 json.loads() - Parse JSON string to Python dict
  • 2 json.dumps() - Convert Python object to JSON string
  • 3 response.json() - Parse API response JSON
Python JSON Examples 
import json
import requests

# Sample JSON data
data = {
    "name": "NITDA ULA",
    "type": "education",
    "features": ["api", "authentication", "security"],
    "active": True

# Convert Python dict to JSON string
json_string = json.dumps(data, indent=2)

response = requests.get("https://api.ula.nitda.gov.ng/v1/data")

    # Access nested data

Complete Example

Python Implementation 
import requests
from requests.exceptions import RequestException
import json

class NITDAAPIClient:
    def __init__(self, client_id, client_secret):
        self.base_url = "https://api.ula.nitda.gov.ng/v1"
        self.client_id = client_id
        self.client_secret = client_secret
        self.access_token = None
        
    def authenticate(self):
        """Get access token"""
        try:
            response = requests.post(
                f"{self.base_url}/auth/token",
                json={
                    "client_id": self.client_id,
                    "client_secret": self.client_secret,
                    "grant_type": "client_credentials"
                },
                timeout=5
            )
            response.raise_for_status()
            self.access_token = response.json()["access_token"]
            return True
            
        except RequestException as e:
            print(f"Authentication failed: {e}")
            return False
    
    def get_user_data(self, user_id):
        """Get user data from API"""
        if not self.access_token:
            if not self.authenticate():
                return None
                
        try:
            response = requests.get(
                f"{self.base_url}/users/{user_id}",
                headers={"Authorization": f"Bearer {self.access_token}"},
                timeout=10
            )
            response.raise_for_status()
            return response.json()
            
        except RequestException as e:
            print(f"API request failed: {e}")
            return None

# Usage
client = NITDAAPIClient("your_client_id", "your_client_secret")
user_data = client.get_user_data("user_123")
if user_data:
    print(json.dumps(user_data, indent=2))
