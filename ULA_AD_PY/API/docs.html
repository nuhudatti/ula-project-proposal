Authentication

Obtaining API Credentials

All API requests require authentication using JWT (JSON Web Tokens). To get started:

  1. Register your application through the NITDA Developer Portal
  2. Receive your API keys (Client ID and Client Secret)
  3. Generate access tokens using the authentication endpoint

Authentication Flow

Here's how to authenticate with the ULA API using Python:

Python Example 
import requests

auth_url = "https://api.ula.nitda.gov.ng/v1/auth/token"
headers = {
    "Content-Type": "application/json"
}
data = {
    "client_id": "your_client_id",
    "client_secret": "your_client_secret",
    "grant_type": "client_credentials"
}

response = requests.post(auth_url, headers=headers, json=data)
access_token = response.json()["access_token"]

Note: Access tokens expire after 1 hour. Use the refresh token to obtain new ones. Store your credentials securely and never expose them in client-side code.

Content Management

GET /courses
Retrieve all available courses

Parameters

Name Type Required Description
page integer No Page number
per_page integer No Items per page (default: 20)

Example Request

headers = {
    "Authorization": f"Bearer {access_token}"
}
response = requests.get(
    "https://api.ula.nitda.gov.ng/v1/courses",
    headers=headers,
    params={"page": 1, "per_page": 10}
