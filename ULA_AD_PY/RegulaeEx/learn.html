üß™ Real Example

Let‚Äôs say we want to pull out all the numbers from a sentence:

text = "We have 3 Apples and 14 Oranges."
pattern = r"\d+"
print(re.findall(pattern, text))

Output: ['3', '14']

We used \d+ where:

  • \d = digit
  • + = 1 or more digits grouped
So the pattern tells Python: ‚ÄúFind me all number blocks in this text.‚Äù

#1 ‚Äì Introduction to Regex
Regular Expressions (regex) let you define search patterns. Python uses the `re` module. 
import re

text = "We have 3 Apples and 14 Oranges."
pattern = r"\d+"
print(re.findall(pattern, text))
['3', '14']
#2 ‚Äì Simple Character Matches
Match specific strings inside text using a simple pattern. 
text = "cat bat mat rat"
pattern = "mat"
print(re.findall(pattern, text))
['mat']
#3 ‚Äì Character Classes
Character classes match groups like letters or digits. 
text = "We have 3 Apples and 14 Oranges."
pattern = r"\d+"
print(re.findall(pattern, text))
['3', '14']
#4 ‚Äì Special Characters
Regex has special characters:
  • ^ = starts with
  • $ = ends with
  • . = any character
  • | = or
text = "Hello!"
pattern = r"^H.*!$"
print(re.findall(pattern, text))
['Hello!']
#5 ‚Äì Quantifiers
Quantifiers allow repetition:
  • * = 0 or more
  • + = 1 or more
  • ? = 0 or 1
  • {n} = exactly n
text = "Heeeeey!"
pattern = r"He{4,}y!"
print(re.findall(pattern, text))
['Heeeeey!']
#6 ‚Äì Substitution
Use `re.sub()` to replace matches with another word. 
text = "Python is powerful. Python is fun."
pattern = "Python"
print(re.sub(pattern, "Regex", text))
Regex is powerful. Regex is fun.
#7 ‚Äì Compiling Patterns
Pre-compile patterns for performance if used multiple times. 
pattern = re.compile(r"\d+")
print(pattern.findall("There are 4 apples and 15 oranges"))
['4', '15']
#8 ‚Äì Splitting Strings
Split text based on patterns using `re.split()`. 
text = "apple,orange;banana|grape"
pattern = r"[,;|]"
print(re.split(pattern, text))
['apple', 'orange', 'banana', 'grape']
