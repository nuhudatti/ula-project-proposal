📘 Age Guess Game – Walkthrough

Step-by-step guide with Python logic

📌 What’s This Project?

This project lets a user enter their birth year and instantly calculates their age using JavaScript (and Python in the backend version). It teaches:

  • User input via forms
  • Calculations using date/time
  • Frontend logic using JavaScript
  • Basic Python logic for age calculation

🧠 How the HTML + JS Works

HTML Form – This is the input form:

<input type="number" id="birthYear" />

JS Logic – This calculates the age:


const birthYear = parseInt(document.getElementById("birthYear").value);
const currentYear = new Date().getFullYear();
const age = currentYear - birthYear;

Then we display it back inside the result element.

🐍 Python Version

Try this in your local Python file or console:


from datetime import datetime

birth_year = int(input("Enter your birth year: "))
current_year = datetime.now().year
age = current_year - birth_year

if birth_year < 1900 or birth_year > current_year:
    print("🚫 Invalid year entered!")
else:
    print(f"🎉 You are approximately {age} years old!")
