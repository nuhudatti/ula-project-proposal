📘 BMI Calculator – Full Walkthrough

🔍 Project Goal

To create a health-conscious web app using HTML, Tailwind CSS, and JavaScript that calculates Body Mass Index (BMI) and offers results based on the user's height and weight.

🛠️ Tools & Skills Used

  • HTML Forms & Semantic Tags
  • Tailwind CSS for UI Design
  • JavaScript DOM Manipulation
  • Conditional Logic & Math Operations

🧩 Code Breakdown

1. HTML Form

<form id="bmiForm">
  <input type="number" id="weight" placeholder="Weight (kg)">
  <input type="number" id="height" placeholder="Height (m)">
  <button>Calculate</button>
</form>

Captures user input for weight and height. The form is connected to a JavaScript function via the ID.

2. JavaScript Function

document.getElementById('bmiForm').addEventListener('submit', function(e) {
  e.preventDefault();
  const weight = parseFloat(document.getElementById('weight').value);
  const height = parseFloat(document.getElementById('height').value);
  const bmi = weight / (height * height);
  // ...

This part reads the inputs, calculates BMI, and shows the result. It also prevents the page from refreshing using e.preventDefault().

3. Logic & Result Display

let message = '';
if (bmi < 18.5) message = 'Underweight';
else if (bmi < 25) message = 'Normal';
else message = 'Overweight';

document.getElementById('result').textContent = `BMI: ${bmi.toFixed(2)} - ${message}`;

Gives a simple health message based on the BMI range. The result is shown on screen using textContent.

🎯 Try This On Your Own

  • Color-code results (green = normal, red = high, yellow = low)
  • Add a reset button to clear inputs
  • Use localStorage to save previous BMI calculations
  • Include gender/age to give more personalized feedback
